# InvestMate Backend 🚀

This is the backend API for InvestMate (Zerodha Clone) built using Node.js, Express, and MongoDB.

---

## 🔧 Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- bcrypt

---

## 📁 Features

- User Registration API
- User Login API
- Password Hashing (bcrypt)
- JWT Token Authentication
- MongoDB Database Integration

---

### 1. Install dependencies

### 2. Start MongoDB
Make sure MongoDB is running locally

### 3. Run server
